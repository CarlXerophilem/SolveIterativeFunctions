Theory of Iterative Functional Equations

1. Function Iteration & Half-Iterates

Function iteration is the process of applying a function repeatedly to its own output. The \(n\)-th iterate of a function \(f\), denoted \(f^{\circ n}\), is defined recursively by \[ f^{\circ 0}(x) = x, \qquad f^{\circ n+1}(x) = f\bigl(f^{\circ n}(x)\bigr). \]

An iterative functional equation asks: given a function \(F(x)\), find \(f(x)\) such that some iterate of \(f\) equals \(F\). The simplest non-trivial case is the functional square root or half-iterate: \[ f^{\circ 2}(x) = f(f(x)) = F(x). \]

This problem sits at the intersection of dynamical systems, functional equations, and enumerative combinatorics. While some equations admit elegant closed-form solutions, others resist any simple expression — and the difference often comes down to the presence or absence of real fixed points.

Definition (Functional Square Root). A function \(f: D \to D\) is a functional square root of \(F: D \to D\) if \(f(f(x)) = F(x)\) for all \(x \in D\). We also say \(f\) is a half-iterate of \(F\).

2. Fixed Points & Their Dynamical Role

A fixed point of a function \(F\) is a value \(x^*\) such that \(F(x^*) = x^*\). Fixed points are the anchors of iteration: they are points where the dynamics stand still.

If \(f\) is a half-iterate of \(F\), then every fixed point of \(f\) is also a fixed point of \(F\). Moreover, any cycle of \(f\) of length 2 becomes a fixed point of \(F\): \[ f(x) = y,\; f(y) = x \quad\Longrightarrow\quad F(x) = f(f(x)) = f(y) = x. \]

Theorem (Necessary condition). If \(F\) has a half-iterate \(f\) on a domain \(D \subseteq \mathbb{R}\), then \(F\) must have either real fixed points or 2-cycles of \(f\) (which are fixed points of \(F\)). In particular, if \(F(x) > x\) for all \(x \in \mathbb{R}\), then \(F\) has no real half-iterate.

Compare the two canonical examples:

\(F(x) = x^2 - 2\)

Fixed points at \(x = -1\) and \(x = 2\).

\(F(x) = x^2 + 1\)

No real fixed points — the graph never meets \(y = x\).

3. A Solvable Case: \(f(f(x)) = x^2 - 2\)

The equation \[ f(f(x)) = x^2 - 2 \] has a well-known closed-form solution: \[ f(x) = x^2 - 2 \quad\text{is its own half-iterate? No — let's check.} \] Actually, \((x^2-2)^2 - 2 = x^4 - 4x^2 + 2 \neq x^2 - 2\).

The true solution connects to Chebyshev polynomials. Recall that \(T_n(\cos\theta) = \cos(n\theta)\). In particular, \[ T_2(x) = 2x^2 - 1, \qquad T_2(T_2(x)) = T_4(x). \] By a linear change of variables, the equation \(f(f(x)) = x^2 - 2\) is solved by \[ f(x) = 2\cos\bigl(\sqrt{2}\,\arccos(x/2)\bigr) \quad\text{for } |x| \le 2, \] or more elegantly, using the conjugacy \(h(x) = x + 1/x\): \[ F(x) = x^2 - 2 = h^{-1}(h(x)^2), \qquad f(x) = h^{-1}\bigl(h(x)^{\sqrt{2}}\bigr). \]

Example: Verifying the solution. Let \(h(x) = x + \frac{1}{x}\) for \(x > 0\). Then \(h^{-1}(y) = \frac{y + \sqrt{y^2-4}}{2}\). Define \(f(x) = h^{-1}\bigl([h(x)]^{\sqrt{2}}\bigr)\). Then \[ f(f(x)) = h^{-1}\bigl([h(f(x))]^{\sqrt{2}}\bigr) = h^{-1}\bigl([h(x)]^{(\sqrt{2})^2}\bigr) = h^{-1}\bigl([h(x)]^2\bigr) = x^2 - 2. \] The critical insight: conjugacy to a power function makes iteration trivial.

4. The Unsolvable Challenge: \(f(f(x)) = x^2 + 1\)

In contrast, the equation \[ f(f(x)) = x^2 + 1 \] has no known closed-form solution on the real line. The fundamental reason is dynamical:

  • \(F(x) = x^2 + 1 > x\) for all real \(x\), so the iterates \(x, F(x), F(F(x)), \ldots\) form a strictly increasing sequence diverging to \(+\infty\).
  • There are no real fixed points, and no real 2-cycles.
  • The equation \(x^2 + 1 = x\) has discriminant \(1 - 4 = -3 < 0\).
Theorem (Obstruction). If \(F: \mathbb{R} \to \mathbb{R}\) satisfies \(F(x) > x\) for all \(x\), then \(F\) has no continuous half-iterate \(f: \mathbb{R} \to \mathbb{R}\).
Proof

Suppose \(f(f(x)) = F(x) > x\) for all \(x\). Consider the orbit of any point \(x_0\) under \(f\). If \(f(x_0) = x_0\), then \(F(x_0) = x_0\), contradicting \(F(x) > x\). If \(f(x_0) > x_0\), then applying \(f\) (which must be increasing if continuous and invertible) gives \(f(f(x_0)) > f(x_0)\), so \(F(x_0) > f(x_0) > x_0\). Repeating, the sequence \(x_0, f(x_0), F(x_0), f(F(x_0)), \ldots\) is strictly increasing. By continuity, \(f\) has no fixed points. But for a continuous function on \(\mathbb{R}\) with no fixed points, we must have \(f(x) > x\) for all \(x\) or \(f(x) < x\) for all \(x\). If \(f(x) > x\) for all \(x\), let \(x_0\) be arbitrary. Then the sequence \(x_n = f^{\circ n}(x_0)\) is strictly increasing. If unbounded, \(f(f(x_n)) = F(x_n)\) grows quadratically, forcing a contradiction with continuity. If bounded, the limit \(L\) satisfies \(f(L) = L\), a fixed point, contradiction. Hence no continuous solution exists on all of \(\mathbb{R}\).

However, numerical solutions exist on restricted domains. The Composita method (Kruchinin & Kruchinin, 2013) provides a systematic way to compute the coefficients of a formal power series for \(f(x)\) satisfying \(f(f(x)) = F(x)\). See the Solver page for an interactive implementation.

5. Conjugacy & the General Picture

Two functions \(F\) and \(G\) are conjugate if there exists an invertible function \(h\) such that \[ F = h^{-1} \circ G \circ h. \] Conjugacy preserves dynamical structure: fixed points, cycles, and iteration counts. If \(G\) is easy to iterate (e.g., \(G(x) = x^\alpha\)), then we can solve \(f(f(x)) = F(x)\) by:

  1. Find \(h\) such that \(F = h^{-1} \circ G \circ h\).
  2. Let \(g(x) = x^{\sqrt{\alpha}}\) (a half-iterate of \(G\)).
  3. Then \(f = h^{-1} \circ g \circ h\) satisfies \(f^{\circ 2} = F\).
Key Insight. The solvability of \(f(f(x)) = F(x)\) hinges on whether \(F\) is conjugate to a simple function (like \(x^\alpha\) or a monomial) on a suitable domain. For \(F(x) = x^2 - 2\), the Chebyshev conjugacy \(h(x) = x + 1/x\) works. For \(F(x) = x^2 + 1\), no such conjugacy exists on \(\mathbb{R}\) because the dynamical behavior — no fixed points, monotonic divergence — cannot be matched by any power function on the real line.

For a more general approach, the Composita of a generating function (see Composita Calculator) provides a combinatorial framework for manipulating iterates and compositions of formal power series, enabling systematic computation even when closed-form conjugacies are unavailable.
