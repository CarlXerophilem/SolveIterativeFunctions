# Web extract — Wikipedia "Functional square root" (half-iterate)

Source: https://en.wikipedia.org/wiki/Functional_square_root (fetched via Chrome, 2026-06-22)

A functional square root (half iterate) of g is f with f(f(x)) = g(x). Notation f = g^{1/2}.

## History (with exact citations)
- **Kneser, H. (1950).** "Reelle analytische Lösungen der Gleichung φ(φ(x)) = e^x und verwandter
  Funktionalgleichungen". *Journal für die reine und angewandte Mathematik* **187**: 56–67.
  doi:10.1515/crll.1950.187.56 — the half-exponential; later basis for non-integer tetration (2017).
- **Babbage (1815)** first studied f(f(x))=x over ℝ (involutions) — **Babbage's functional equation**.
  Particular solution f(x) = (b − x)/(1 + c x) for bc ≠ −1. Babbage noted any conjugate
  Ψ^{-1}∘f∘Ψ is again a solution ⇒ the group of invertible functions acts on the solution set by conjugation.

## Solutions
- Systematic functional n-roots (real, negative, infinitesimal n) of g:ℂ→ℂ rely on solutions of
  **Schröder's equation**. Infinitely many trivial solutions when the root's domain is enlarged.

## Examples
- f(x)=2x^2 is a functional square root of g(x)=8x^4.
- Half-iterate of the Chebyshev polynomial T_n: f(x)=cos(√n · arccos x) (generally not a polynomial).
- f(x)=x/(√2 + x(1−√2)) is a functional square root of g(x)=x/(2−x).
- sin^{1/2}(1) ≈ 0.90871.
