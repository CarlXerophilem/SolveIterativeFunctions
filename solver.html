Iterative Function Solver

This calculator now separates two complementary viewpoints on the equation \(f(f(x)) = F(x)\). Composita / Paper Mode works with truncated power series and extends the current site machinery toward the paper-backed family \(A^{\circ 2^m}(x) = F(x)\). Analytic Fixed-Point Mode studies a chosen real fixed point and, when the multiplier is suitable, constructs a local half-iterate through Schröder linearization.

The second mode is intentionally local: it shows where analytic theory is stable and where a real, lightweight browser tool should stop and warn instead of inventing a global closed form.

1. Choose a Solver Mode

Uses the triangular composita recurrence to compute a series candidate \(A(x)\) such that \(A^{\circ 2^m}(x) \approx F(x)\) to the chosen truncation order.

2. Define the Target Function \(F(x)\)

Enter coefficients for the polynomial model \(F(x) = c_0 + c_1 x + c_2 x^2 + \cdots + c_6 x^6\). Paper mode requires \(c_0 = 0\) and \(c_1 > 0\). Analytic mode accepts a nonzero constant, but the chosen \(x_0\) must still satisfy \(F(x_0) = x_0\).

Presets: \(x + x^3\) \(x + x^2\) \(2x + x^2\) \(x + 2x^2 + x^3\)
Presets: Attracting \(0.5x + 0.1x^2\) at \(0\) Repelling \(2x + x^2\) at \(0\) \(x^2 - 2\) at \(2\) Parabolic \(x + x^2\) at \(0\)
