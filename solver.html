Iterative Function Solver

This page solves the functional equation \(A(A(x)) = F(x)\) for a given polynomial \(F(x) = f_1 x + f_2 x^2 + f_3 x^3 + \cdots\) using the Composita recurrence (Kruchinin & Kruchinin, 2013). The solution \(A(x) = a_1 x + a_2 x^2 + \cdots\) is computed term-by-term and verified graphically.
